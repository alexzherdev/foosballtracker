# Foosball Tracker
A simple app to keep track of your foosball match scores, at home or in the office game room.

Will support storing 1v1, 2v2, and even 1v2 results, plus numerous stats.

## Technologies Used

* React.js v15
* Flux
* Gulp, Browserify, Babel, ESLint
* ES6
